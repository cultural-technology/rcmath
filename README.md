# rcmath
*Rust Cryptographic Math Library*

#### Core
- [x] Big unsigned integer (**uint**): Like: U256, U512, U1024, U2048, U4096.
- [x] Finite field (**ff**)
- [x] Group (**group**)
- [x] Pairing curves (**pairing**)
- [x] Fast Fourier transform (**fft**)

#### Auxiliary
- [x] no-std
- [x] parallel
- [ ] serde
- [ ] simd
- [ ] asm
